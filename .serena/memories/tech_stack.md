# Tech Stack

- Backend: Python + FastAPI.
- Frontend: HTML, CSS e JavaScript puro.
- Armazenamento inicial: arquivo JSON, temporario, nao banco profissional.
- Nao usar React nesta primeira versao.
- Nao usar banco profissional nesta primeira versao; JSON pode ser substituido futuramente por SQLite, PostgreSQL ou AWS RDS.
- Ferramentas previstas: VS Code, Codex, ESP32 enviando dados para servidor local.
